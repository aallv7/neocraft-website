Siapakah Kami?

Kami adalah tim yang terdiri dari beberapa orang yang senang bermain Minecraft seperti kamu!

Server NeoCraft awalnya dimulai saat 2 orang menemukan cara untuk membuat server di Minecraft Bedrock edition. Lalu, mereka mengajak teman-temannya untuk bermain bersama. Mereka pun mencoba membuat server Minecraft Java edition, dan menemukan sistem plugin dan juga plugin yang menghubungkan pemain Java edition dan Bedrock edition di satu server. Dan saat itulah terbentuk IklanMC yang merupakan server pertama yang dibangun oleh kami.

Pada masa itu, IklanMC masih memiliki spesifikasi rendah dan akhirnya performanya tidak bagus. IklanMC mengalami perubahan nama menjadi Gado-Gado SMP setelah beberapa waktu tidak online. Gado-Gado SMP pun memiliki nasib yang sama dengan IklanMC, yang pada akhirnya kami memilih spesifikasi server yang bagus dan melalukan rebranding menjadi NeoCraft.
