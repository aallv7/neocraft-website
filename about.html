Siapakah Kami?

Kami adalah tim yang terdiri dari beberapa orang yang senang bermain Minecraft seperti kamu!

Server NeoCraft awalnya dimulai saat 2 orang menemukan cara untuk membuat server di Minecraft Bedrock edition. Lalu, mereka mengajak teman-temannya untuk bermain bersama. Mereka pun mencoba membuat server Minecraft Java edition, dan menemukan sistem plugin dan juga plugin yang menghubungkan pemain Java edition dan Bedrock edition di satu server. Dan saat itulah terbentuk IklanMC yang merupakan server pertama yang dibangun oleh kami.

Pada masa itu, IklanMC masih memiliki spesifikasi rendah dan akhirnya performanya tidak bagus. IklanMC mengalami perubahan nama menjadi Gado-Gado SMP setelah beberapa waktu tidak online. Gado-Gado SMP pun memiliki nasib yang sama dengan IklanMC, yang pada akhirnya kami memilih spesifikasi server yang bagus dan melalukan rebranding menjadi NeoCraft.

Peraturan

Sebelum kamu bergabung, jangan lupa membaca peraturannya ya!

Peraturan Komunitas


  1. Mohon berperilaku sopan

    2. Perlakukan semua orang dengan hormat. Sama sekali tidak ada pelecehan, seksisme, rasisme, atau ujaran kebencian yang akan ditoleransi.


  2. Dilarang mengirim ataupun membicarakan hal-hal yang melanggar asusila

  3. Patuhi tim Administrator dan Moderator kami

    4. Meminta bantuan tentang server dimohon untuk sopan dan tidak secara spam. Administrator dan Moderator tidak segan menghukum bagi yang melanggar dan yang tidak patuh.


  4. Dilarang spam atau promosi diri

    5. Tidak ada spam atau promosi diri (undangan server, iklan dll) tanpa izin dari anggota staf.



Peraturan Server


  1. Dilarang metode duping apapun

    2. "Loh, katanya ini server semi-anarki?" Iya, betul. Tetapi, sudah tertera ini adalah server semi-anarki dan tidak sepenuhnya anarki. Kami memiliki sistem ekonomi yang dimana mudah runtuh jika duping diperbolehkan. Tapi tenang saja, di server ini tidak akan ada sistem pay to win apalagi perjudian (crate, dkk).


  2. Dilarang melakukan segala sesuatu yang menyebabkan gangguan pada server

    3. Seperti, lag machine, chunk loader, dan farm berlebih-lebihan.


  3. Dilarang melakukan segala sesuatu yang membuat pemain lain tidak bisa memasuki server

    4. Seperti, chunkban dan lain sebagainya.
